a basic windows PIN rectool (recovery/recreational tool)

- features customizable creation time in ms
- char length
- output to file (log.txt)

discovery :
i discovered this after wanting to modify my windows lockscreen with a malformed .png with a custom alias that pointed to a .gif (or originally .mp4) 
to play on the lockscreen, the mp4 did not work, but the .gif caused a black screen-> ok? cool?
so this did not work, so i then decided "what if i can open the background settings directly from the lockscreen?"
and i then updated the custom alias, pointed to a .exe file (which was a compiled panel to display an embedded image), named OSK.exe, then restarted. 
  the osk was now my custom application.
    and so this was born.
